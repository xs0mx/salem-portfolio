Clarkson SDIP REU: VR Rehabilitation Coach

A research-focused VR rehabilitation system that combines Unity development, movement tracking, exercise feedback, and structured data logging for upper-limb rehabilitation research.

Project Objective

The goal is to develop a Unity-based VR rehabilitation application for upper-limb rehabilitation. The user picks up a virtual dumbbell in VR, performs a bicep curl, and receives real-time feedback on movement quality.

The summer target is one polished exercise done well: a clean bicep curl workflow with a grabbable dumbbell, movement tracking, rep counting, basic scoring, and data export for later lab analysis.

Core System Features

  • Clean VR rehabilitation room scene.
  • Working bicep curl exercise with a grabbable virtual dumbbell.
  • Movement tracking using the Quest headset and controllers.
  • Repetition counting and a basic movement-quality score.
  • Real-time visual feedback with green/yellow/red indicators.
  • Short text cues, rep counter, and research-ready movement logging.
Technical Plan

How the system is structured

VR Interaction

Build a rehab room scene in Unity with a virtual dumbbell that can be grabbed and moved during the exercise.

Movement Tracking

Use headset and controller movement to estimate the user’s exercise motion and track bicep curl performance.

Feedback System

Display color-coded feedback, short guidance messages, and a rep counter to make the exercise interactive and understandable.

Quality Score

Calculate a basic score based on movement quality so the user receives a simple measure of how well the rep was performed.

Data Export

Log movement information in a research-ready format, such as CSV, so it can be reviewed or analyzed after a session.

Research Value

The project connects VR, rehabilitation, data collection, and applied AI interests in a system that can support future research work.
