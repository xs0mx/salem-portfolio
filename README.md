# Salem Nasereddin Portfolio Website

This is a multi-page dark-mode professional portfolio website built with plain HTML, CSS, and JavaScript.

## Included pages
- `index.html` - Home
- `about.html` - About and education
- `projects.html` - Selected projects
- `research.html` - Clarkson SDIP REU / VR Rehabilitation Coach
- `experience.html` - Experience and skills
- `contact.html` - Contact links

## Assets
- `assets/profile.jpg` - Profile photo
- `assets/Salem_Nasereddin_Resume.pdf` - Sanitized resume with phone number and full address hidden
- `assets/internship-presentation.mp4` - Compressed internship presentation video
- `assets/video-thumbnail.jpg` - Video thumbnail

## How to run locally
Open `index.html` in a web browser.

## How to deploy
You can upload the full folder to GitHub Pages, Netlify, or Vercel. For GitHub Pages, push these files to a repository and enable Pages from the repository settings.
